# Task Decomposition
Decomposing end-to-end Goal into a text analytics pipeline.

# Launching
## Back-end 
1. Set up virtual environment 
2. run `pip install -r server/requirements.txt`
3. run `python -m server.main`

## Front-end
1. run `npm i`
2. run `npm run dev`
